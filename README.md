## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/portfolio-site-using-github-pages.git
cd portfolio-site-using-github-pages
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

---

## 🌍 Deployment (GitHub Pages)

```bash
npm run deploy
```
